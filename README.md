# Smart Mirror

## Installation

You will need:

*   Java 8
*   npm (which is installed with [node.js](https://nodejs.org/))

## Running the application

Run the following:

1.  `npm install -g bower`
2.  `bower install`
3.  `./gradlew run`

The application should now be on [http://localhost:9098](http://localhost:9098)
